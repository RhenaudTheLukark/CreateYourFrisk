The Overworld: The Map Object


Each map has its own settings, which are determined by the mod it loads, the music it plays, and a bunch of other stuff.
Listed below are functions mainly used to access these variables.


Map.GetMusic() Returns the name of the music the map launches when the player teleports to this map.


Map.SetMusic(string value) Sets the name of the music the map launches when the player teleports to this map.


Map.GetModToLoad() Returns the name of the mod the map uses.


Map.SetModToLoad(string value) Sets the name of the mod the map uses. The player needs to enter the map again for this new mod to be loaded.


Map.GetNoRandomEncounter() Returns the value of the boolean NoRandomEncounter. The boolean NoRandomEncounter prevents encounters from being launched if set to true.


Map.SetNoRandomEncounter(bool value) Sets the value of the boolean NoRandomEncounter.


Map.GetMusicKept() Returns the value of the boolean isMusicKeptBetweenBattles. This one is pretty straightforward, if set to true, the map's music won't be stopped during battles, like in The Core.


Map.SetMusicKept(bool value) Sets the value of the boolean isMusicKeptBetweenBattles. If you change the value of this boolean, the main Audio channel used for the map's bgm will be changed (and all of Audio's functions will be used on this new channel), being "scr" when this boolean's value is true, and "StaticKeptAudio" otherwise.


Map.GetMusicMap(string mapName) Returns the name of the music the map given launches when the player teleports to this map. If you choose to access the current map's MusicKept value, it'll launch Map.GetMusic() instead.
ONLY WORKS WHEN THE PLAYER HAS ALREADY ENTERED THE MAP BEFORE! Use Map.HasPlayerBeenInMap() to check if they did.


Map.SetMusicMap(string mapName, string value) Sets the name of the music the map given launches when the player teleports to this map. Also works on maps the player hasn't been in yet. If you choose to modify the current map's Music value, it'll launch Map.SetMusic() instead.


Map.GetModToLoadMap(string mapName) Returns the name of the mod the map given uses. If you choose to access the current map's ModToLoad value, it'll launch Map.GetModToLoad() instead.
ONLY WORKS WHEN THE PLAYER HAS ALREADY ENTERED THE MAP BEFORE! Use Map.HasPlayerBeenInMap() to check if they did.


Map.SetModToLoadMap(string mapName, string value) Sets the name of the mod the map uses. Also works on maps the player hasn't been in yet. If you choose to modify the current map's ModToLoad value, it'll launch Map.SetModToLoad() instead.


Map.GetNoRandomEncounterMap(string mapName) Returns the value of the boolean NoRandomEncounter of the given map. The boolean NoRandomEncounter prevents encounters from being launched if set to true. If you choose to access the current map's NoRandomEncounter value, it'll launch Map.GetNoRandomEncounter() instead.
ONLY WORKS WHEN THE PLAYER HAS ALREADY ENTERED THE MAP BEFORE! Use Map.HasPlayerBeenInMap() to check if they did.


Map.SetNoRandomEncounterMap(string mapName, bool value) Sets the value of the boolean NoRandomEncounter in the given map's data. Also works on maps the player hasn't been in yet. If you choose to modify the current map's NoRandomEncounter value, it'll launch Map.SetNoRandomEncounter() instead.


Map.GetMusicKeptMap(string mapName) Returns the value of the boolean isMusicKeptBetweenBattles in the given map's data. This one is pretty straightforward, if set to true, the map's music won't be stopped during battles, like in The Core. If you choose to access the current map's MusicKept value, it'll launch Map.GetMusicKept() instead.
ONLY WORKS WHEN THE PLAYER HAS ALREADY ENTERED THE MAP BEFORE! Use Map.HasPlayerBeenInMap() to check if they did.


Map.SetMusicKeptMap(string mapName, bool value) Sets the value of the boolean isMusicKeptBetweenBattles in the given map's data. Also works on maps the player hasn't been in yet. If you choose to modify the current map's MusicKept value, it'll launch Map.SetMusicKept() instead.


Map.HasPlayerBeenInMap(string map) Returns a boolean specifying whether the player has been in that map before or not. If they did, the function will return true, false otherwise.
