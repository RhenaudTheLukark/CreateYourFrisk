The Discord object [E/M/W] This class is used to manipulate the data the Discord Rich Presence Status bar displays, if Discord is on and if this feature is enabled.

  • Discord.SetName( name = "", noPrefix = false): Sets the top row of the Discord Rich Presence Status bar, if it's enabled. If you are in battle, it will display Playing Mod: at the beginning of the line, unless noPrefix is set to true, in which case name will be displayed as-is.

    By default, this line displays Playing Mod: MODNAME, with MODNAME being the name of your mod.

    Setting name to its default value "" doesn't display the line at all.

  • Discord.SetDetails( details = ""): Sets the second row of the Discord Rich Presence Status bar, if it's enabled.

    By default, this line isn't displayed.

    Setting details to its default value "" doesn't display the line at all.

  • Discord.SetTime( time = 0, isCountdown = false): Sets the current time displayed in the Discord Rich Presence Status bar, if it's enabled. The time value is displayed in seconds. The isCountdown value tells the engine whether this time must be counted down or not, allowing it to act much like a stopwatch if it's enabled.

    By default, this line will display the time elapsed since the mod was started.
